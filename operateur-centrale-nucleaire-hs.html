  • Le statut IEG n'est pas le Code du travail standard — les règles HS sont définies par les accords EDF.
  • Les syndicats EDF (CGT Énergie, CFDT Énergie) connaissent parfaitement ces règles spécifiques.
  • Les arrêts de tranche sont planifiés longtemps à l'avance — les HS devraient être prévues dans le planning.
  • Les prestataires travaillant sur site EDF (sous-traitants) relèvent du Code du travail classique.
  • En cas de litige, le tribunal judiciaire est compétent (pas le CPH) pour les agents au statut IEG.

Prestataires sur site nucléaire : le droit commun s'applique

  • Les sous-traitants travaillant sur les centrales nucléaires relèvent du Code du travail et de leur CCN.
  • Ils ne bénéficient pas du statut IEG mais ont droit aux mêmes protections de radioprotection.
  • Leurs HS se calculent selon leur propre CCN (BTP, maintenance industrielle, etc.).
  • L'exposition aux rayonnements ionisants pendant les HS doit être déclarée et comptabilisée.
  • La dosimétrie officielle peut servir de preuve des heures passées sur site.

Questions fréquentes

Le statut IEG d'un opérateur de centrale nucléaire suit-il le Code du travail classique pour les heures supplémentaires ?

Non, le statut des Industries Électriques et Gazières (IEG) n'est pas le Code du travail standard — les règles d'heures supplémentaires sont définies par les accords spécifiques d'EDF, distincts du droit commun.

En cas de litige sur mes heures supplémentaires au statut IEG, dois-je saisir les prud'hommes ?

Non, pour les agents au statut IEG, c'est le tribunal judiciaire qui est compétent en cas de litige, pas le Conseil de prud'hommes — une différence importante à connaître avant d'engager une procédure.
