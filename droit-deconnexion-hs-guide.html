Renard Juriste confirme
Le droit a la deconnexion (art. L2242-17 Code du travail) impose une negociation annuelle dans les entreprises de plus de 50 salaries. Mais l absence d accord ne cree pas un droit de solliciter les salaries indefiniment - les HS generees restent dues.

Preuves et recours pour les HS de deconnexion

  • Exportez regulierement en PDF vos emails professionnels tardifs - l horodatage est incontestable.
  • Les logs de connexion au VPN sont accessibles via demande RGPD (reponse en 1 mois).
  • Les messages WhatsApp professionnels peuvent etre produits comme preuves (Cass. 2021).
  • Si votre charte interdit la connexion apres 19h mais votre manager vous ecrit a 22h : double violation.
  • Documentez pendant 3 mois avant d agir : un echantillon representatif suffit pour etayer.
