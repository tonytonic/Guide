Points clés à retenir

En immobilier, les visites et signatures se font souvent hors horaires normaux :

  • Visites le samedi (et parfois le dimanche) : journées souvent de 10h
  • Compromis de vente : souvent le soir après 18h pour les acquéreurs
  • Assemblées générales de copropriété : souvent en soirée
  • Permanences téléphoniques : peuvent constituer de l'astreinte

Les commissions des négociateurs immobiliers entrent dans la base de calcul des HS (taux horaire moyen = fixe + commissions du mois).

Questions fréquentes

Je suis négociateur sans forfait jours. Mes samedis de visites génèrent-ils des HS ?

Oui si la semaine dépasse 35 h. Un samedi de 10h ajouté à une semaine de 30h (lundi-vendredi) = 40h → 5 HS.

Mes commissions entrent-elles dans le calcul de mes HS ?

Oui. Les commissions sur ventes entrent dans le taux horaire moyen servant de base aux HS.
