Renard Expert
Dans la restauration rapide, les forfaits jours sont souvent invalides faute d'entretiens annuels. Le turn-over élevé entraîne l'omission systématique de cet entretien obligatoire. Si vous avez été manager sans entretien annuel : 3 ans de HS réclamables.

Points de vigilance spécifiques

  • Les heures de nettoyage avant ouverture ou après fermeture = TTE.
  • Les briefs d'équipe imposés hors horaires = TTE → HS si dépassement.
  • Les CDD étudiants ont les mêmes droits aux HS que les CDI.
  • Les données de caisse (horaires ouverture/fermeture) prouvent les heures de présence.
  • Le contingent CCN restauration rapide est de 220h/an.
Renard Juriste · confirmé
Dans les grandes chaînes (McDonald's, Burger King, KFC), les données du logiciel de caisse sont horodatées à la seconde. Ces données — obtenues via demande RGPD — constituent des preuves irréfutables des horaires réels.
