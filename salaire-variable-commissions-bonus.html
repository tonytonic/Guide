Salaires variables et bonus 2026 : impact sur les heures supplémentaires

Tes HS se calculent sur ton taux horaire réel — y compris les commissions et les primes. Si tu es payé en variable, le calcul de tes HS est plus complexe mais toujours dû.

Le cadre légal

Le calcul des heures supplémentaires en présence d'un salaire variable est régi par l'article L3121-22 du Code du travail et la jurisprudence de la Cour de cassation.

Renard Omniscient
La jurisprudence Cass. soc. 2023 a confirmé que les commissions sur ventes entrent dans la base des HS. Si ton employeur calcule tes HS uniquement sur le fixe, il te doit un rappel sur 3 ans.

Points clés à retenir

Principe fondamental : les HS se calculent sur le taux horaire réel, pas uniquement sur le salaire de base. Les éléments variables de rémunération entrent dans la base si :

  • Ils rémunèrent du travail effectif (commissions sur ventes, primes de production)
  • Ils ne compensent pas un risque particulier (primes de danger, d'insalubrité)
  • Ils ne sont pas expressément exclus par la jurisprudence (participation, intéressement)

Pour calculer le taux horaire moyen incluant les variables : (salaire fixe + variables du mois) ÷ heures contractuelles du mois = taux horaire moyen. Les HS sont majorées de 25 % ou 50 % sur ce taux.

Questions fréquentes

Je suis commercial avec salaire fixe + commissions. Mes HS sont-elles calculées sur le fixe seulement ?

Non. La Cour de cassation est constante : les commissions entrent dans la base de calcul des HS car elles rémunèrent du travail effectif. Le taux horaire moyen (fixe + commissions) ÷ heures travaillées s'applique.

Mon bonus annuel entre-t-il dans le calcul des HS de l'année ?

Si ce bonus rémunère les performances de l'année (objectifs atteints), oui. Il faut le proratiser sur les 12 mois et l'ajouter à la base de calcul mensuelle des HS. C'est complexe mais réclamable.

Je suis en forfait jours avec partie variable. Mon forfait est-il valide ?

La partie variable peut coexister avec un forfait jours valide. Mais le forfait doit être expressément prévu par la CCN et le contrat, avec suivi de charge. Si le forfait est nul, le taux horaire moyen (fixe + variable) sert de base aux HS réclamables.
