Chiffres indicatifs 2026. Seul votre bulletin de paie fait foi.

  • Le taux horaire de base doit inclure tous les éléments réguliers de rémunération (primes, commissions).
  • Les deux taux de majoration (+25 % et +50 %) ne s'appliquent jamais au même taux horaire de base — recalculez à chaque palier.
  • La réduction de cotisations salariales (11,31 %) s'applique automatiquement sur le bulletin — vérifiez la ligne dédiée.
  • Le plafond de 7 500 €/an est calculé sur le cumul de toutes les HS de l'année, tous employeurs confondus.
  • Au-delà du plafond, seul le surplus redevient imposable — la réduction de cotisations reste acquise.

Questions fréquentes

Quels éléments de rémunération dois-je inclure dans le taux horaire de base pour calculer l'exonération ?

Tous les éléments réguliers de rémunération (primes, commissions habituelles) doivent être inclus dans le taux horaire de base — pas seulement le salaire fixe.

Le plafond de 7 500 € d'exonération s'apprécie-t-il par employeur ou pour l'ensemble de mes revenus ?

Il est calculé sur le cumul de toutes tes heures supplémentaires de l'année, tous employeurs confondus. Au-delà du plafond, seul le surplus redevient imposable — la réduction de cotisations salariales (11,31 %) reste acquise.
